# Higher-Lower-Game
A CLI-based game implemented in Python that challenges users to compare follower counts of famous entities.

## Implementation Notes
* **Data Management:** Handles data stored in dictionaries within a list (nested structures).
* **Game Logic:** Implements an infinite loop that breaks upon an incorrect user guess.
* **Modularity:** Divided into separate files (`art.py`, `game_data.py`) for cleaner code organization.

## Usage
Execute the main script to start the game:
`python Higher_or_Lower_game.py`

## Core Concepts Used
* Python Dictionaries & Lists.
* Conditional logic and input validation.
* Modular programming.
